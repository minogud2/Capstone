INTRODUCTION
------------

This is the repository for the Capstone project: "Expanding the Secondary School Network in Sub-Saharan Africa
using Meta-Heuristic Facility-Location Techniques". Five jupyter notebooks are included in this repository. 
Two notebooks dedicated to data pre-processing are included in the pre_processing folder: a) pre_processing, and
b) calculate_distance. Another notebook 'initial_data_exploration' provides some analysis of the current datasets.
The experiments are run on 'main' and the results are cleaned and summarised in the 'Results' notebook. Each
notebook provide more details within. 

REQUIREMENTS
------------

In order to run this project you will need python and the packages specified in the requirements.txt file.
